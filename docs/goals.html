Goal targets

Target gas values we want clients to reach for the key 8038 parameters, checked against each client's latest estimate. A client clears a goal when its estimate is at or below the target — lower is faster.

  • The account goals (COLD_ACCOUNT_ACCESS, ACCOUNT_WRITE) require a client to clear both the CODE and NOCODE variants — both values are shown in the cell.
  • The write goals subtract the bundled cold-access component before comparing: STORAGE_WRITE uses COLD_STORAGE_WRITE − 3000 and ACCOUNT_WRITE uses COLD_ACCOUNT_*_WRITE − 3000 (the access goals).
